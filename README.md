❤️ AI Based Heart Disease Detection System

This project is a simple machine learning based web application that predicts the likelihood of heart disease using medical parameters.

Overview

The system uses a Random Forest model trained on the Cleveland Heart Disease dataset. Users can enter details like age, cholesterol, blood pressure, etc., and get an instant prediction.

Features
• Predicts heart disease risk using machine learning  
• Simple and user-friendly web interface  
• Fast and real-time results  
• Easy to use for basic health analysis  

Technologies Used
• Python  
• Flask  
• HTML, CSS  
• Scikit-learn  
• Pandas  

How It Works
• User enters medical details  
• Data is sent to backend (Flask)  
• Model processes the input  
• Result is displayed:  
  • Heart Normal  
  • Heart Disease Likely  

Model Details
• Algorithm: Random Forest  
• Dataset: Cleveland Heart Disease Dataset (Kaggle)  
• Accuracy: ~75%  

Deployment

The project is deployed using Render and the source code is available on GitHub.

Note

This project is for educational purposes only and not a substitute for medical advice.
