❤️ AI Based Heart Disease Detection System

A simple and effective Machine Learning + Web Application that predicts the likelihood of heart disease based on user-provided medical data.

🚀 Overview

This project uses a Random Forest Machine Learning model to analyze important health parameters such as age, cholesterol level, blood pressure, and more.

It is integrated with a Flask-based web application that allows users to enter medical details and get instant prediction results.


💡 Key Features

🔍 Heart Disease Prediction using Machine Learning
🌐 User-Friendly Web Interface for easy input
⚡ Real-Time Results displayed instantly
🧠 AI-Based Analysis of medical parameters
📱 Responsive Design for smooth usage



🛠️ Technologies Used

Python – Core programming language
Flask – Backend web framework
HTML & CSS – Frontend design
Scikit-learn – Machine learning model
Pandas – Data processing



📊 Machine Learning Model

Algorithm: Random Forest Classifier
Dataset: Cleveland Heart Disease Dataset (Kaggle)
Accuracy: ~75%



⚙️ How It Works

User opens the web application
Enters medical details (age, cholesterol, BP, etc.)
Data is sent to Flask backend
Model analyzes the input
Prediction is generated:
✅ Heart Normal
⚠️ Heart Disease Likely
Result is displayed instantly.
